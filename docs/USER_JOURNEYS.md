# YETO User Journeys

## Overview

This document maps the key user journeys through the YETO platform, identifying touchpoints, actions, and expected outcomes for each user persona.

---

## User Personas

### 1. Policy Analyst (Sarah)
- **Role**: Economic analyst at international organization
- **Goals**: Research Yemen economy, produce policy briefs
- **Tech Level**: Intermediate
- **Frequency**: Weekly

### 2. Journalist (Ahmed)
- **Role**: Business reporter covering Yemen
- **Goals**: Find data for stories, verify claims
- **Tech Level**: Basic
- **Frequency**: As needed

### 3. Humanitarian Worker (Maria)
- **Role**: Program manager at NGO
- **Goals**: Track food security, plan interventions
- **Tech Level**: Basic
- **Frequency**: Daily

### 4. Researcher (Dr. Hassan)
- **Role**: Academic economist
- **Goals**: Comprehensive data for research papers
- **Tech Level**: Advanced
- **Frequency**: Monthly

### 5. Government Official (Fatima)
- **Role**: Ministry of Planning staff
- **Goals**: Monitor indicators, prepare reports
- **Tech Level**: Intermediate
- **Frequency**: Daily

## Journey 1: First-Time Visitor Exploration

**Persona**: Any new visitor

### Steps

```
Homepage → Dashboard → Sector Page → Data Point → Evidence Pack
```

| Step | Page | Action | Outcome |
|------|------|--------|---------|
| 1 | Homepage | View hero, stats, CTA | Understand platform purpose |
| 2 | Homepage | Click "Explore Dashboard" | Navigate to main dashboard |
| 3 | Dashboard | View key indicators | See economic overview |
| 4 | Dashboard | Toggle regime (Aden/Sana'a) | Compare dual authorities |
| 5 | Dashboard | Click sector link | Navigate to sector detail |
| 6 | Sector | View indicator detail | Understand specific metric |
| 7 | Sector | Click "Show Evidence" | View data provenance |
| 8 | Evidence | Review sources | Verify data quality |

### Success Metrics
- Time to first insight: < 2 minutes
- Pages viewed: 4+
- Return visit within 7 days
