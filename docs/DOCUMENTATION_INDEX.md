### For Data Sources
```markdown
# Source Name

## Overview
What data this source provides.

## Coverage
Geographic scope, time period, update frequency.

## Reliability
Confidence rating, known limitations.

## Access
How to access (API, download, etc.), authentication required.

## Ingestion
How data is ingested into YETO, transformation steps.

## Licensing
License type, attribution requirements.
```
