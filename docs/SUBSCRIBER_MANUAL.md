# YETO Platform Subscriber Manual

## Welcome to YETO

The Yemen Economic Transparency Observatory (YETO) provides comprehensive economic data, analysis, and intelligence on Yemen's economy. This manual guides you through using the platform effectively.

---

## Getting Started

### Creating an Account

1. Visit the YETO platform homepage
2. Click "Sign In" in the header
3. Authenticate via Manus OAuth
4. Your account is automatically created on first login

### Subscription Tiers

| Tier | Access Level | Key Features |
|------|--------------|--------------|
| **Public** | Basic | Dashboard overview, limited exports |
| **Registered** | Standard | Full dashboards, basic exports, saved searches |
| **Pro** | Advanced | API access, custom reports, priority support |
| **Institutional** | Enterprise | Bulk data, white-label, dedicated support |

## Using the Dashboard

### Regime Toggle

Yemen's split governance requires tracking data separately:
- **Both**: View data from both authorities side-by-side
- **Aden (IRG)**: Internationally Recognized Government data only
- **Sana'a (DFA)**: De Facto Authority data only

### Time Period Selection

Adjust the date range to focus on specific periods:
- Use the calendar picker for custom ranges
- Select granularity: Annual, Quarterly, or Monthly

### Exporting Data

Click the "Export Data" button to download:
- **CSV**: For spreadsheet analysis
- **JSON**: For programmatic use
- **Excel**: For formatted reports

## Sector Dashboards

Each sector page provides:

1. **Key Metrics**: Top-line indicators with trends
2. **Interactive Charts**: Time series with regime comparison
3. **Data Tables**: Detailed values with sources
4. **Related Reports**: Relevant publications and analysis

### Understanding Confidence Levels

| Level | Meaning | Visual |
|-------|---------|--------|
| A | High confidence - Official, verified | Green badge |
| B | Medium-high - Reputable source | Blue badge |
| C | Medium - Single source, plausible | Yellow badge |
| D | Low - Estimated or contested | Red badge |

## AI Assistant ("One Brain")

### Asking Questions

The AI assistant can answer questions about Yemen's economy:
- "What is the current exchange rate in Aden?"
- "How has inflation changed since 2020?"
- "Compare banking sector performance between regimes"

### Understanding Responses

Each AI response includes:
- **Answer**: The main response text
- **Confidence Score**: How certain the AI is (0-100%)
- **Sources**: Data sources used for the answer
- **Related Indicators**: Relevant metrics to explore

### Limitations

- AI only answers based on YETO data (RAG-only)
- May decline questions outside its knowledge base
- Always verify critical information with primary sources

## Best Practices

### Data Interpretation

1. **Always check confidence levels** - Higher confidence = more reliable
2. **Compare both regimes** - Get the full picture
3. **Review sources** - Understand where data comes from
4. **Check dates** - Ensure data is current

### Citation

When using YETO data, cite as:
```
Source: Yemen Economic Transparency Observatory (YETO)
URL: https://yeto.causewaygrp.com
Accessed: [Date]
```

### Data Limitations

- Some indicators have gaps due to conflict
- Regime-specific data may not be comparable
- Historical data quality varies
